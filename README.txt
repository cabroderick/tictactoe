Tic Tac Toe
By Collin Broderick

This repository contains a Tic-Tac-Toe / Connect Four solver. The board is represented in board.txt. 'x' represents an x, 'o' represents an o, and empty spaces are represented by a space. There must be exactly 4 lines containing 4 characters each for the solver to run.

To run:
python main.py
To run unit tests:
python -m unittest tests.py
